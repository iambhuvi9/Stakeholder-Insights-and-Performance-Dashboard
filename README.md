# Stakeholder-Insights-and-Performance-Dashboard
Power BI dashboard analyzing stakeholder performance, premiums and profitability insights.

# Project Overview
This project presents an interactive Power BI dashboard designed to analyze stakeholder performance, premiums, assets under management and profitability metrics.

The dashboard helps organizations understand value creation and financial performance across stakeholders.

# Objectives
- Analyze equity value and premium performance
- Track assets under management
- Understand stakeholder demographics
- Evaluate cost ratio vs profitability
- Monitor premiums over time

# Tools Used
- Power BI
- Excel
- DAX

# Key KPIs
- Equity Value Created: 230.82bn
- Avg Stakeholder Age: 34.46
- Total Premium: 245.91bn
- Highest Value Created: 49.17M
